# Cafe Sales Analysis

## Project Overview

This project analyzes cafe sales data using Python, Pandas, Matplotlib, and Seaborn.

## Objectives

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sales Performance Analysis
- Payment Method Analysis
- Location Analysis
- Time Series Analysis

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights

- Highest Revenue Item
- Most Popular Payment Method
- Best Sales Month
- Top Revenue Location

## Visualizations

- Bar Chart
- Pie Chart
- Line Chart
- Heatmap
- Box Plot
- Histogram
